# Brick Lister - Aplikacja React (SPA)

Projekt zrealizowany przy użyciu Vite i React.js. Wykorzystuje `react-router-dom` do obsługi nawigacji oraz React Hooks (`useState`) do zarządzania stanem komponentów.

## Wymagania:
* Zainstalowane środowisko Node.js (wraz z menedżerem pakietów `npm`).
*Uwaga: Funkcjonalności odczytu/zapisu plików CSV oraz generowania PDF zostały pominięte zgodnie z wytycznymi (przekreślenia w instrukcji).*
